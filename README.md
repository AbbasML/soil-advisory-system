# 🌾 Soil Advisory System — Team CodeHarvest

AI-Powered Soil Health Analysis and Advisory System with Intelligent Chatbot for Farmers

## Project: TechForGood 2026 — IEEE Student Branch MIT ADT University

### Features

- Soil health analysis using pH and NPK values
- Nutrient deficiency detection
- Crop suitability recommendation
- Fertilizer recommendation with exact doses
- AI chatbot (KisanBot) for farmer guidance
- Multilingual support (English, Hindi, Marathi, Gujarati)

### Tech Stack

- Frontend: React + Vite
- Backend: Python + FastAPI
- AI: Google Gemini API
- Deployment: Vercel + Render
